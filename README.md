# 🧠 Programming Paradigms — Java

This repository contains projects developed for the Programming Paradigms course, focused on applying object-oriented programming principles and software architecture patterns.

---

## 🧩 Key Concepts
- 🏗️ Object-Oriented Design (encapsulation, abstraction, modularity)
- 🧭 Model-View-Controller (MVC) architecture
- ⚡ Event-driven programming
- 🔄 State management and game loops
- 🎮 Input handling and rendering

---

## 👾 Main Project: Ms Pacman
A 2D tile-based game built from scratch in Java.

### ✨ Features
- 🎯 Player movement and collision detection
- 📷 Scrolling camera system
- 🖼️ Sprite animation and asset management
- 🧱 Separation of game logic and rendering using MVC
- ⌨️ Keyboard event handling

---

## 🎯 Purpose
The goal of these projects is to practice writing maintainable and scalable code by separating responsibilities and organizing large programs into clear components.
