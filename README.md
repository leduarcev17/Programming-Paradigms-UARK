# 🧠 Programming Paradigms

This repository contains projects and assignments developed for the Programming Paradigms course at the University of Arkanasas in Spring of 2026, exploring different programming languages, software design patterns, and problem-solving approaches across multiple paradigms.

Throughout the semester, projects were implemented in **Java, JavaScript, and Python**, focusing on object-oriented programming, MVC architecture, event-driven systems, functional programming concepts, and game development.

# 👾 Main Project — Ms Pacman Engine

The primary project of this course was a custom-built 2D Ms Pacman-style game engine recreated across multiple languages during the semester.

## ☕ Java Version
Focused on strong OOP design and MVC architecture.

### Features
- 🎯 Player movement and collision handling
- 👻 Multiple ghost entities with animations
- 🍒 Fruit entities with autonomous movement
- 📷 Scrolling camera system
- 🖼️ Sprite animation system
- 🧱 Tile-based map editor
- 💾 Save/load system using JSON
- ⌨️ Keyboard and mouse input handling
- 🔄 Polymorphic sprite system using inheritance

---

## 🌐 JavaScript Version
Reimplemented the engine in JavaScript to explore scripting languages and dynamic programming concepts.

### Features
- ⚡ Browser-based game loop
- 🧠 Functional programming techniques
- 🎮 Event-driven interactions
- 🔄 Timed animations and asynchronous behavior
- 🧩 Dynamic object handling and loose typing

---

## 🐍 Python Version
Rebuilt the project using Python and Pygame to explore scripting languages and duck typing.

### Features
- 🎮 Pygame rendering and input system
- 🧠 Duck Typing polymorphism
- 👾 Sprite management and collisions
- ⏱️ Timed state transitions and animations
- 🔄 Dynamic object behavior without inheritance requirements

---

## 🎯 Course Goals
The purpose of these projects was to understand how different programming paradigms influence software design and development.

This repository demonstrates:
- Writing maintainable and modular code
- Separating responsibilities through architecture patterns
- Applying polymorphism in statically and dynamically typed languages
- Comparing compiled vs scripting languages
- Building scalable interactive applications across multiple environments
